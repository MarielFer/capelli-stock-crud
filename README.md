# CapelliStock - CRUD de Productos Capilares

## Descripción
Aplicación web desarrollada en ASP.NET Core MVC que permite gestionar un inventario de productos capilares mediante operaciones CRUD (Crear, Leer, Actualizar y Eliminar).

## Tecnologías utilizadas
- C#
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- Git & GitHub

## Funcionalidades
- Registro de productos
- Edición de productos
- Eliminación de productos
- Listado de productos
- Validación de datos en formularios

## Metodología Git Flow
El proyecto fue desarrollado utilizando Git Flow con las siguientes ramas:

- main
- develop
- qa
- feature/home-design
- feature/ui-buttons
- feature/search-product
- feature/ui-improvements
- feature/validation-alerts

Se realizaron un total de 15 Pull Requests correctamente integrados.

## Ejecución del proyecto
1. Clonar el repositorio
2. Abrir en Visual Studio
3. Configurar la cadena de conexión en appsettings.json
4. Ejecutar el proyecto
